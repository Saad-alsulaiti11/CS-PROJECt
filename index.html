Exploring Cloud Computing, Data Transmission, and Internet Technologies

Cloud Computing & Web Apps

Cloud computing allows data to be stored and processed on remote servers instead of your local device. HTML, CSS, and JavaScript run on the client-side (your browser), while cloud servers handle storage and processing on the server-side.

Internet Technologies

DNS Server & URL/IP Relationship

When you type www.google.com in your browser, a DNS server translates it into an IP address (like 142.250.190.78). The browser then sends a request through multiple routers to reach the server. The server responds and the website is displayed.

Broadband Connection: Fiber Optic

Fiber optic broadband uses light signals to transmit data quickly over long distances. It is ideal for streaming, online gaming, and large file transfers.

Cloud vs Traditional Storage

Traditional storage keeps data on local devices like hard drives or USBs. Cloud storage keeps data on remote servers accessible online.

Cloud storage allows collaboration, remote access, automatic backups, and scalable storage without buying extra hardware.
